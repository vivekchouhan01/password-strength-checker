# 🔐 Password Strength Checker

A modern and responsive **Password Strength Checker** built with **HTML, CSS, and JavaScript**. It analyzes the strength of a password in real time and provides instant visual feedback to help users create stronger passwords.

---

## 🚀 Features

- ✅ Real-time password strength detection
- 👁️ Show/Hide password toggle
- 🎨 Responsive and modern UI
- ⚡ Instant strength indicator
- 🔴 Weak Password
- 🟡 Moderate Password
- 🟢 Strong Password
